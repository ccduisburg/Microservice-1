# Microservice Project


### Tech Stack 
 - Java 11
 - Spring Framework
 - Spring Boot
 - Spring Cloud
 - Spring Data
 - REST-API
 - API-Gateway
 - Config-server
 - Eureka-server
 - Eureka-client
 - Admin-server
 - Zipkin-server
 - Netflix-Hystrix
 - Cassandra
 - Elasticsearch
 - MySQL
 - RabbitMQ
 - Actuator
 - Fabric8
 - Docker
 - Lombok
 
### Requirements

For building and running the application you need:
- [JDK 11](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) or newer . 
- [Maven](https://maven.apache.org)
- [Docker](https://www.docker.com/)
- [Lombok](https://projectlombok.org/)
